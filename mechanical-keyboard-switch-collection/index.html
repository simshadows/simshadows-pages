Mechanical Keyboard Switch Collection

This page is a visual guide on what switches I own in my collection. It's made for my own benefit to help me identify switches, but I made it a public webpage since there's no reason it can't be used by anyone else!

If you think I've made any mistakes, please open an issue on GitHub.

These images are not my own. They were pulled from various places on the internet. I choose images according to what I think best-represents the switch appearance in-person. For original image sources, click on the images.

Some columns are provided more to assist in telling apart different switches:

  • Top Label: Moulded label at the top of the switch. These values are case-sensitive to best-represent the font used for the label.
  • SMD: SMD LED compatibility.
    • If Cutout, then there is a cutout to allow light to shine through.
    • If Transparent, then there is no cutout but light should be able to shine through.
    • If Semi-Transparent, then there is no cutout and light might be able to shine through, but may be more heavily-attenuated than a truly transparent housing.
    • If No, then there is no cutout and the bottom housing is opaque, which isn't designed to let light through.
  • Pins: Either 3-pin or 5-pin.

I have also included notes that I've observed that are helpful for identifying visually identical samples in my collection. These observations are expected to not generalize (e.g. mould numbers).
