Sim's Mechanical Keyboard Switch Collection

This page is a visual guide on what switches I own in my collection. It's made for my own benefit to help me identify and compare, but I made it a public webpage since there's no reason to keep it to myself. I hope you will find this useful!

For now, I only collect stock MX-style switches. I'll update this if I get around to lubing, filming, or building frankenswitches.

If you think I've made any mistakes, please open an issue on GitHub.

These images are not my own. They were pulled from various places on the internet. I choose images according to what I think best-represents the switch appearance in-person. For original image sources, click on the images.

Some columns are provided more to assist in telling apart different switches:

  • Top Label: Moulded label at the top of the switch. These values are case-sensitive to best-represent the font used for the label.
  • SMD: SMD LED compatibility.
    • If Cutout, then there is a cutout to allow light to shine through.
    • If Transparent, then there is no cutout but light should be able to shine through.
    • If Semi-Transparent, then there is no cutout and light might be able to shine through, but may be more heavily-attenuated than a truly transparent housing.
    • If No, then there is no cutout and the bottom housing is opaque, which isn't designed to let light through.
  • Pins: Either 3-pin or 5-pin.
  • Additional Identification Notes: Any other interesting features.

Disclaimers:

  • The column Interesting Documented Characteristics is based entirely on found documentation. Some possible issues:
    • Semantic consistency: Terms used can mean different things between different sources. If I feel a term is well-defined, it will be typeset as strong. If I feel a term is vaguely defined or imprecise, it will be typeset as emphasis. Unintuitive but well-defined terms I will use to the best of my ability:
      • Operating force: Peak force required to reach the operating position.
      • Pretravel: Distance between rest and the operating position.
      • Total travel: Distance between rest and bottom-out.
      • (Operating position: Position in which the switch registers a change of signal.)
    • Documentation consistency: It is assumed that the design documentation is accurately reflected in reality. It is also assumed that the documentation is free of typos.
